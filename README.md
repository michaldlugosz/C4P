# Consult4Procure - strona na GitHub Pages

## Jak opublikować stronę za darmo

1. Załóż konto na GitHub.com.
2. Utwórz nowe repozytorium, np. `consult4procure`.
3. Wgraj do repozytorium pliki:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
4. Wejdź w `Settings` -> `Pages`.
5. W sekcji `Build and deployment` wybierz:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Kliknij `Save`.
7. Po chwili strona będzie dostępna pod adresem:
   `https://NAZWA-UZYTKOWNIKA.github.io/consult4procure/`

## Co zmienić przed publikacją

W pliku `index.html` zmień:

- adres e-mail: `kontakt@consult4procure.pl`
- numer telefonu: `+48 000 000 000`
- ewentualnie zakres usług i treści biznesowe

## Własna domena

Hosting GitHub Pages jest darmowy. Własna domena, np. `consult4procure.pl`, jest opcjonalna i zwykle wymaga zakupu domeny u rejestratora.
